Task 14: Palindromes

Write a JavaScript function findPalindromes(value) that extracts from a given text all palindromes, e.g. "ABBA", "lamal", "exe". Write JS program palindromesExtract.js that invokes your function with the sample input data below and prints the output at the console. Examples:

Some texts:

The car was there!

There is a man, his name was Bob.

ABBA, lamal, exe, Ana, This, anna, or elle, even eve, best hanah, may Hannah
